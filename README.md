# EntainSudokuFrontend And EntainSudokuBackend

This project contains both a frontend and a simple server for testing Sudoku game with multiplayer mode!

## Dependencies

First install dependencies:

```bash
npm install
```

## Development command for Frontend + Server

Now start the frontend and the backand with this simple command.

```bash
npm run dev
```

## Enjoy

Have in mind that in order to be able to test the multiplayer mode you need to log in from a separate session
(e.g. from incognito mode or different browser or device).
